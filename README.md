# TaskReward サンプルサイト

TaskReward（業務委託型タスク報酬SaaS）のUIデモサイト。

**全データは模擬データ**で、DB接続なしで動作します。デザイン・遷移フローの確認用。

## 機能

- ランディングページ（LP）
- ログイン / 新規登録画面（フォーム表示のみ）
- ダッシュボード（残高・累計報酬・デイリーミッション）
- 案件一覧 / 案件詳細（カテゴリフィルタ・ソート）
- ミッションフロー（動画視聴 → クイズ → CTA）
- ランキング
- 紹介プログラム
- 出金履歴
- 利用規約 / プライバシーポリシー

## 技術スタック

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- lucide-react

## ローカル起動

```bash
npm install
npm run dev
```

http://localhost:3000 で起動します。

## デプロイ

Vercelに接続してmainブランチへpushで自動デプロイ。環境変数の設定は不要です。

## 注意

- このリポジトリはUIデモ用です。本番データベース・認証・決済機能はありません。
- 本番サイトは別リポジトリで運用しています。
